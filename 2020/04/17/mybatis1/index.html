mybatisжәҗз Ғз¬”и®°пјҲдёҖпјү

еҸ‘еёғдәҺ 2020-04-17 жәҗз Ғ java mybatis

mybatisжҳҜзҺ°еңЁжөҒиЎҢзҡ„жҢҒд№…еұӮжЎҶжһ¶пјҢзҺ°еңЁиҝҷе®¶е…¬еҸёд№ҹд»Һд№ӢеүҚзҡ„sshйЎ№зӣ®ж…ўж…ўиҪ¬жҲҗдәҶssmйЎ№зӣ®пјҢиҖҢжҲ‘д№ӢеүҚз”ЁиҝҮibatisпјҢibatisжӯЈжҳҜmybatisзҡ„еүҚиә«гҖӮж•…жғіжүҫж—¶й—ҙжқҘйҳ…иҜ»дёӢе…¶жәҗз ҒпјҢдҪҶжҳҜжҜҸеӨ©иғҪжҠҪеҮәжқҘзҡ„ж—¶й—ҙжңүйҷҗпјҢеҸӘиғҪж…ўж…ўзҡ„и®°еҪ•дёӢжқҘгҖӮ

жәҗз Ғи°ғиҜ•

mybatisжҳҜдёҖдёӘејҖжәҗзҡ„жЎҶжһ¶пјҢжүҖд»ҘжҲ‘д»¬зӣҙжҺҘд»ҺgithubдёҠжӢүеҸ–д»Јз ҒпјҢз”ҡиҮіеҸҜд»ҘforkеҲ°иҮӘе·ұзҡ„д»“еә“йҮҢпјҢд№ҹж–№дҫҝиҮӘе·ұеңЁдёҠйқўж·»еҠ жіЁйҮҠгҖӮ
жҲ‘дҪҝз”ЁIDEAејҖеҸ‘е·Ҙе…·жӢүеҸ–д»Јз ҒпјҢзүҲжң¬дёә3.5.3гҖӮ
жӢүеҸ–е®ҢжҲҗеҗҺеҸҜд»ҘжүҫдёӢorg.apache.ibatis.autoconstructor.AutoConstructorTest

AutoConstructorTestжҳҜеҚ•е…ғжөӢиҜ•зұ»пјҢд»»ж„ҸдёҖдёӘеҚ•е…ғжөӢиҜ•ж–№жі•йғҪеҸҜд»ҘзӣҙжҺҘејҖе§Ӣи°ғиҜ•гҖӮе®ғдҪҝз”Ёзҡ„жҳҜHSQLDBгҖӮ

ејҖе§Ӣйҳ…иҜ»

Resources.getResourceAsReader()жҳҜиҜ»еҸ–xmlй…ҚзҪ®гҖӮ
иҝӣе…Ҙnew SqlSessionFactoryBuilder().build(),дјҡеҸ‘зҺ°жңүеҫҲеӨҡbuild()жҳҜжңүеҫҲеӨҡйҮҚиҪҪзҡ„ж–№жі•пјҢдҪҶжңҖз»ҲйғҪдјҡеҺ»жү§иЎҢbuild(Reader reader, String environment, Properties properties)пјҢжүҖд»ҘзӣҙжҺҘзңӢиҝҷдёӘеҘҪдәҶгҖӮ
еңЁжӯӨж–№жі•йҮҢпјҢдјҡеҲӣе»әдёҖдёӘXMLConfigBuilderпјҢзӣҙжҺҘзӮ№иҝӣеҺ»зңӢе…¶жһ„йҖ еҮҪж•°гҖӮ
е®ғжңүеҫҲеӨҡжһ„йҖ еҮҪж•°пјҢйҡҸдҫҝжҢ‘дёҖдёӘжқҘзңӢгҖӮ

иҝҷйҮҢжңүдёӨдёӘйҮҚиҰҒзҡ„еҜ№иұЎпјҢдёҖдёӘжҳҜXPathParserе’ҢXMLMapperEntityResolverгҖӮ
XPathParserжҳҜmybatisеҹәдәҺXPathи§ЈжһҗеҷЁпјҢе°ҒиЈ…дәҶXPathгҖҒDocumentе’ҢEntityResolverпјҢз”ЁжқҘи§Јжһҗй…ҚзҪ®ж–Үд»¶mybatis-config.xmlд»ҘеҸҠMapper.xmlпјҢеұһдәҺmybatisзҡ„еҹәзЎҖж”ҜжҢҒеұӮдёӯзҡ„и§ЈжһҗеҷЁжЁЎеқ—гҖӮXMLMapperEntityResolverжҳҜEntityResolverзҡ„е®һзҺ°зұ»пјҢж ёеҝғжҳҜresolveEntity()ж–№жі•пјҢжқҘиҜ»еҸ–DTDж–ҮжЎЈгҖӮDTDж–Үд»¶дё»иҰҒйӘҢиҜҒxmlж–Үд»¶зј–еҶҷзҡ„еҗҲжі•жҖ§гҖӮ
жҹҘзңӢXPathParserзҡ„жәҗз ҒпјҢйҰ–е…Ҳе®ғжңүеҫҲеӨҡжһ„йҖ еҮҪж•°пјҢд№ҹйҡҸдҫҝжҢ‘дёҖдёӘжқҘзңӢпјҢ

commonConstructor()дёәеҲқе§ӢеҢ–XPathParserзҡ„еҮ дёӘеҸӮж•°гҖӮ
createDocument()жҳҜе°Ҷxmlи§ЈжһҗжҲҗDocumentеҜ№иұЎгҖӮ
еңЁи°ғз”ЁcreateDocument()ж–№жі•еүҚпјҢдёҖе®ҡиҰҒе…Ҳи°ғз”ЁcommonConstructor()гҖӮ

еӣһеҲ°XMLConfigBuilderжһ„йҖ еҮҪж•°дёӯжқҘгҖӮ

жҺҘдёӢжқҘзңӢзңӢXMLConfigBuilder.parse()ж–№жі•гҖӮ

parsedиЎЁзӨәxmlжҳҜеҗҰе·Із»Ҹи§ЈжһҗиҝҮдәҶ гҖӮ
parseConfiguration()ж–№жі•еҲҷжҳҜи§Јжһҗ XMLдёӯзҡ„configurationиҠӮзӮ№гҖӮ
иҝӣе…ҘparseConfiguration()гҖӮ

и§Јжһҗе®Ңxmlиҝ”еӣһConfigurationеҜ№иұЎгҖӮ
жңҖз»ҲSqlSessionFactoryBuilder.bulid()иҝ”еӣһSqlSessionFactoryеҜ№иұЎгҖӮ
иҮіжӯӨз¬¬дёҖйғЁеҲҶд»Јз Ғжү§иЎҢз»“жқҹпјҢеү©дёӢзҡ„йғЁеҲҶе°ұжҜ”иҫғз®ҖеҚ•дәҶгҖӮ

д»ҺsqlSessionFactoryдёӯиҺ·еҸ–еҲ°ж•°жҚ®еә“иҝһжҺҘпјҢе№¶жү§иЎҢжҢҮе®ҡи·Ҝеҫ„дёӢзҡ„sqlж–Үд»¶пјҢдёәжөӢиҜ•зұ»з”ҹжҲҗжөӢиҜ•ж•°жҚ®гҖӮ

еҘҪдәҶпјҢд»ҘдёҠе°ұжҳҜжҳҜз®ҖеҚ•зҡ„еҠ иҪҪmybatis-configж–Үд»¶жөҒзЁӢгҖӮе…¶дёӯXMLConfigBuilderдёӯиҝҳжңүдёҖдәӣж–№жі•пјҢж”ҫеҲ°еҗҺйқўеңЁзңӢгҖӮ
